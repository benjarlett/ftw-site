The Workshop accessible video player

Simple example

This page shows an example of how to embed the The Workshop Flash video player (v0.1.0.2) and also describes how to embed the player into an HTML page.

We recommend using SWFObject embed, and Flash player version 10.1 and above.

Please see the HTML code of this page to see a working example

To view this content, you must have JavaScript enabled, and the latest version of the Adobe Flash Player.
