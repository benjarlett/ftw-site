The Workshop accessible video player

JavaScript API

Introduction

To use Javascript with the videoplayer you must incluse the 'videoplayer.js' in the <head> of your HTML page.
e.g. 

<script src="videoplayer.js" type="text/javascript"></script>

Referencing the player

When the videoplayer loads it will try to call the function onPlayerInitiated in 'videoplayer.js'.
The video player passes an 'id' to the function. This 'id' must be set in the embed of the player as a FlashVar.
e.g. 

flashvars.id = "video";	//set ID of flash for javascript communication

Where the 'id' is the id of the div that will contain the video player.
Once the video has loaded and the onPlayerInitiated has been called, you will be able to call the following javascript functions to control the video player.

API Methods

Change the video file 

changeVideo("new url")
a link to change the video would be <a href="Javascript:changeVideo("a new url")">Play</a>

Change the caption file 

changeCaptions("new url")
a link to change the captions would be <a href="Javascript:changeCaptions("a new url")">Play</a>

Change the Audio Description file 

changeAudioDescription("new url")
a link to change the audio description would be <a href="Javascript:changeAudioDescription("a new url")">Play</a>

Change the poster image 

changeImage("new url")
a link to change the image would be <a href="Javascript:changeImage("a new url")">Play</a>

Start playback 

play()
a link to start playback would be <a href="Javascript:play()">Play</a>

Pause playback

pause()

a link to stop playback would be <a href="Javascript:pause()">Pause</a>

Skip 2 seconds forward

fastforward()
a link to fast forward would be <a href="Javascript:fastforward()">Fast Forward</a>

Skip 2 seconds back

rewind()
a link to rewind would be <a href="Javascript:rewind()">Rewind</a>

Resart the video

restart()
a link to restart would be <a href="Javascript:restart()">Restart</a>

Toggle Mute on/off

togglemute()
a link to mute/unmute would be <a href="Javascript:togglemute()">Toggle Audio</a>

Increase Volume by 1

volumeup()
a link to increase volume would be <a href="Javascript:volumeup()">Increase Volume</a>

Decrease Volume by 1

volumedown()
a link to decrease volume would be <a href="Javascript:volumedown()">Decrease Volume</a>

Toggle Captions on/off

togglecaptions()
a link to toggle captions would be <a href="Javascript:togglecaptions()">Toggle Captions</a>

Toggle shortcut screen on/off

togglehelp()
a link to toggle the help page would be <a href="Javascript:togglehelp()">Toggle Help</a>

Toggle Audio Description on/off

toggleaudiodescription()
a link to oggle audio description would be <a href="Javascript:toggleaudiodescription()">Toggle Audio Description</a>

Change the text size - cycles through the different sizes

changetextsize()
a link to change the text size would be <a href="Javascript:changetextsize()">Change text size</a>

Change the text colour - cycles through the different colours

changetextcolour()
a link to change the colour settings would be <a href="Javascript:changetextcolour()">Change text colour</a>
