The Workshop accessible video player

Troubleshooting

Step 1: Is the player visible on your page?

To check this, right click over the area where the player should be on you page. If you get a menu telling you it is Flash then the video player is being displayed.

NO

  • Check the URL to the video player swf
  • Check that there are no JavaScript errors

YES

  • Got to the Step 2

Step 2: Can you see the video player controls?

If the player has loaded and initialised correctly then you will be able to see the video player controls

NO

  • Check the video player XML assets (videoplayer_styles.xml and video_player.xml) are all present. Use a debugger such as Firebug to see if you have any 404 errors
  • If you are using a lang setting, check the URLs are correct

YES

  • Go to Step 3

Step 3: Does the media play?

If the player has not loaded the media correctly you will see a spinning circle in the centre of the player. This means that the player is still trying to load the media file.

NO

  • Check the URLs to the media file. Use a debugger such as Firebug to see if you have any 404 errors
  • Check that the media is encoded in the correct format, flv, mp4, mp3

YES

  • You are up and running

Captions/audio description not working?

If the player has not loaded a captions or audio description file correctly the player will not stop working, it will fail silently.

  • Check the URLs to the file. Use a debugger such as Firebug to see if you have any 404 errors
  • Check there is no error in the caption XML file. Open the XML file in a browser and it will highlight any errors

Poster image not showing?

If the player is not showing the poster image the player will not stop working, it will fail silently.

  • Check the URLs to the file. Use a debugger such as Firebug to see if you have any 404 errors
