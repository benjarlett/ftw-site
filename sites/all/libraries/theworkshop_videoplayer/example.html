How to embed the player

Step 1: Include the SWFObject JavaScript in the HTML page

for example within the HTML head tag.

<script src="swfobject/swfobject.js" type="text/javascript"></script>

Step 2: Include the videoplayer.js JavaScript in the HTML page

for example within the HTML head tag.

<script src="videoplayer.js" type="text/javascript"></script>

Step 3: Call the createVideoPlayer() JavaScript function from the HTML body tag

for example in the HTML body tag.

<body onload="createVideoPlayer('video')">

Step 4: Create a HTML div tag with the id attribute "video". The content of this div tag will be replaced with the video player.

For example

<div id="video">The video player will replace this text. You would normally place your no JavaScript message here. For more info see div.</div>

Step 5: In the HTML body tag, set up the Javascript SWFObject embed and the function call createVideoPlayer().

For example

<script type="text/javascript">
	
    //id = is passed to the function and is the id of div that will contain the video player (in this case 'video'). see <body> tag
    function createVideoPlayer(id) {
    
        //change to the width/height of your video
        var playerwidth=600; 
        var playerheight=400;               
        
        var params = {};
        params.allowFullScreen = true;//allows full screen to work
        
        var flashvars = {};	
        flashvars.id = id;	//set ID of flash for javascript communication
        
        //toolbar functionality
        flashvars.hideToolbarWhenPlaying = "false";	//true = hides toolbar when the video is playing [optional default = true]
        
        //The video file
        flashvars.mediaURL = "video/dummy_video.flv";	//FLV file [make sure the URL to the video is relative to the HTML page embedding the player]
        
        flashvars.imageURL = "assets/images/dummy_still.jpg"; //[optional]
        flashvars.captionsURL = "assets/en/captions/dummy_captions.xml"; //[optional]
        flashvars.audioDescriptionURL = "assets/audio/dummy_audiodesc.mp3"; //[optional]
        flashvars.language = "en/";	//folder in assets containing xliff xml [optional]
      
        //video and image will be scaled to the dimensions supplied
        swfobject.embedSWF("assets/videoplayer.swf", id, playerwidth , playerheight , "10.1", null, flashvars, params);
}
</script>

Embed dimensions

FLVs

The player should be embedded at the exact height of the FLV in pixels. To find out the dimensions of the FLV video, use an FLV player such as VideoLan VLC player.

YouTube videos

When embedding the player using a YouTube video, find out what sizes the video you have chosen is available in. Do this by looking at the embed code of your chosen video on the YouTube website.
Typical YouTube dimensions are 560 x 340, 640 x 385, 853 x 505, or 1280 x 745.

playerwidth = the width in pixels of the video
playerheight = the height in pixels of the video

Embed parameters

The standard parameters to use in the SWFObject embed are:

params.bgColor = "#FFFFFF"; // background colour of the embed (optional)
params.allowFullScreen = true; // set to true to enable full screen video playback

Flashvars


Flashvar Function Default Required
id The id of the HTML div where you want the video player to appear. This is required when targeting the video player with JavaScript. This is passed to the function from the function call, ie <body onload="createVideoPlayer('flash')">. N/A Yes
mediaURL The path of the media to be played relative to the SWF file (videoplayer.swf). Or the absolute path to the media (if on another server for example). This is also where you add the id file of your YouTube video eg Oe3St1GgoHQ the last part of the YouTube url. N/a Yes
imageURL The path of image to display when the player loads (PNG, JPG) relative to the HTML page that player is embedded in.
Note: To display correctly, the image should be the same dimensions as the embed dimensions of the player.		 N/a No
captionsURL URL to the captions file (XML) relative to the HTML page that player is embedded in. N/a No
audioDescriptionURL URL to the audio description file (MP3) relative to the HTML page that player is embedded in. N/a No
language Path to the folder containing the containing xliff xml file. This is relative to the SWF file (videoplayer.swf). N/a No
useToolbar This shows or hides the control toolbar. True No
useTextControls This shows or hides the text colour and size option buttons. True No
useFullScreen This shows or hides the full screen button. True No
useShortCuts This shows or hides the help button. True No
useVolume This shows or hides the volume controller and mute button. True No
useRestart This shows or hides the restart button. True No
hideToolbarWhenPlaying This shows or hides the toolbar when the video is playing. True No
hideToolbarWhenPaused This shows or hides the toolbar when the video is paused. False No
captionsOnByDefault Controls the state of the captions when the player first loads, true = captions on. False No
audioDescOnByDefault Controls the state of the audio description when the player first loads, true = audio description on. False No

Please note:

Any combination of FlashVars can be sent in any order. If the URL to the asset is wrong, or the asset file does not exist the video player will ignore the FlashVar sent eg if the Caption file sent does not exist, the player will not display the captions or the captions button.

Container div

The div that will contain the video player must have an id attribute. The function call, createVideoPlayer('video'), most likely in the body tag <body onload="createVideoPlayer('id')">, must include the id of the div where you want the player to be placed on the page. We also recommend that the div should contain a message to tell the user if they do not have the correct Flash player or have JavaScript disabled in their browser. An example including a suitable message is below.
eg

<div id="video">
	<p>To view this content, you must have JavaScript enabled, and the latest version of the <a href="http://www.adobe.com/go/getflashplayer">Adobe Flash Player>/a>.</p>
	<p><a href="http://www.adobe.com/go/getflashplayer"><img src="http://www.adobe.com/images/shared/download_buttons/get_flash_player.gif" alt="Get Adobe Flash Player" border="0" /></a></p>
</div>

Video player help page

We recommend that when embedding the video player on a page, you provide a link to the Video Player Help page. This page contains instructions on how to use the player and details shortcut keys.

Files required by the video player

There are a number of files required by the video player for it to work correctly. These are configuration XML files (project_styles.xml, xliff_loader.xml and the 'xliff' folder and obviously the 'videoplayer.swf' itself) that you will find in the assets folder in this zip file. These files and folder MUST sit in the same location and same level as the videoplayer.swf.
